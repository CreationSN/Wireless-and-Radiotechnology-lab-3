1. **flask_stream.png** - Flask streaming server running on Laptop A
2. **yolo_detection.png** - YOLO detection window on Laptop B with bounding boxes

1. Take screenshots during testing

3. Commit and push to GitHub

### Screenshot Naming Convention:
- `laptop_a_flask_stream.png` - Flask app running
- `laptop_b_yolo_detection.png` - YOLO detection window
- `detected_objects_console.png` - Console output
